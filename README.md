## ğŸ§  Agent Architecture

The system is built using multiple cooperating agents:

1. **Resume Analyzer Agent**
   - Extracts skills, education, interests, and experience from resume text

2. **Career Decision Agent**
   - Matches user profile with potential career roles
   - Generates career match scores

3. **Roadmap Planner Agent**
   - Identifies missing skills
   - Creates a structured learning roadmap

4. **Feedback & Readiness Agent**
   - Calculates career readiness score
   - Supports continuous improvement logic

The agents work together to simulate a real career mentor.
